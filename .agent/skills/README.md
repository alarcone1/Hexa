# Habilidades del Agente Hexa

Este directorio contiene las habilidades personalizadas para el asistente de desarrollo del proyecto Hexa.

## Habilidades Disponibles

1.  **[git-commit-formatter](./git-commit-formatter/SKILL.md)**: Ayuda a formatear los mensajes de commit siguiendo el estándar de *Conventional Commits*.
    - **Solicitudes Comunes:**
        - "Prepara un commit para mis cambios"
        - "Escribe un mensaje de commit convencional"
        - "Confirma los cambios realizados siguiendo el estándar"

2.  **[license-header-adder](./license-header-adder/SKILL.md)**: Automatiza la adición de encabezados de licencia (Apache 2.0 en español) a los archivos del proyecto.
    - **Solicitudes Comunes:**
        - "Añade el encabezado de licencia a este archivo"
        - "Asegúrate de que los nuevos archivos tengan la licencia"
        - "Incluye los derechos de autor en el código"

3.  **[guardian-estilo](./guardian-estilo/SKILL.md)**: Garantiza que todos los cambios visuales sigan estrictamente el archivo `ESTILO.md`.
    - **Solicitudes Comunes:**
        - "Modifica la UI siguiendo el manual de estilo"
        - "Verifica que el nuevo componente cumpla con ESTILO.md"
        - "Ajusta los colores para que coincidan con la guía de diseño"

## Cómo Usar
Para usar una habilidad, el agente lee las instrucciones en el archivo `SKILL.md` correspondiente y sigue los pasos allí descritos.
